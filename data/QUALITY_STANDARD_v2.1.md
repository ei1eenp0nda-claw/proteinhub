# ProteinHub 高质量笔记生成标准 v2.1 - 强调论点与思路

## 核心目标
让一个**没看过原文的研究员**看完笔记后，能：
1. 理解论文的**核心论点**（作者想证明什么）
2. 掌握**论证思路**（作者如何一步步证明）
3. 了解**关键逻辑转折**（从什么发现推导到什么结论）
4. 知道**研究在领域中的位置**（解决了什么长期争议/空白）

---

## 禁止事项 ❌

### 1. 模板化语言
- ❌ "姐妹们！今天来聊..."
- ❌ "记得关注我～"
- ❌ "该领域正在向精准医疗方向发展"

### 2. 空洞概括
- ❌ "AMPK调控能量代谢"（太宽泛）
- ❌ "通过实验发现"（什么实验？）
- ❌ "发挥重要作用"（什么作用？）

### 3. 数据堆砌
- ❌ 列出所有实验数据而没有解释其论证价值
- ❌ "磷酸化水平增加了3.5倍"（如果没有说明这个数据证明了什么论点）

---

## 强制要求 ✅

### 【标题】
- 必须体现**核心发现或机制**
- 示例：✅ "CIDEA通过两亲性螺旋结合磷脂酸促进脂滴融合的分子机制"
- 反例：❌ "CIDEA研究新进展"

### 【研究背景】
**核心：作者为什么要做这项研究？**

必须包含：
1. **领域共识**：这个领域大家已经知道什么（2-3句话）
2. **长期争议/未解问题**：有什么矛盾现象或机制空白？（1-2句话）
3. **本文切入点**：作者从什么角度/用什么工具来解决这个问题（1句话）

示例结构：
```
已知：脂滴融合是脂肪储存的关键，但分子机制不明。
争议：早期研究提出孔道模型和膜融合模型，但都无法解释方向性脂质转移。
切入：作者假设CIDE蛋白可能通过特殊的膜结合机制介导融合。
```

### 【核心发现】强调"论点-证据-逻辑"

每一点必须按以下结构：

```
### 论点N：[一句话概括这个发现]

**作者想证明什么**：
[1-2句话说明核心主张]

**论证思路**：
1. [第一步实验/观察，验证什么假设]
2. [第二步，排除什么替代解释]
3. [第三步，建立什么因果关系]

**关键数据**（支撑核心论点时提供）：
- [关键实验结果，带具体数值]
- [对照实验结果]

**逻辑链条**：
[从A发现如何推导出B结论，中间的关键推理步骤]
```

### 【临床/生物学意义】
**核心：这个发现改变了我们对什么的理解？**

- 对**现有理论**的修正/补充
- 对**疾病机制**的新解释
- 对**治疗策略**的启示

### 【原文信息】
```
标题: [完整英文标题]
期刊: [期刊名] 年份: [YYYY] 引用: [次数]
DOI: [10.xxxx/xxxxx] PMID: [8位数字]
第一/通讯作者: [姓名] 机构: [大学/研究所]
```

### 【一句话总结】
**核心：用最简洁的语言概括"作者证明了什么"**

必须包含：
- 研究对象
- 核心机制/发现
- 生物学意义

### 【延伸阅读】
- 2-3篇**直接相关**的论文
- 说明每篇与本文的关系（支持/对比/延伸）

## 优秀示例要点

### CIDEA论文（eLife 2015）的论点拆解

**核心论点**：
CIDEA通过两亲性螺旋结合磷脂酸（PA），形成跨脂滴同源二聚体，利用PA的锥形结构产生的膜应力促进脂滴融合。

**论证思路**：
1. **定位问题**：CIDEA为什么能待在脂滴上？→ 发现两亲性螺旋
2. **功能问题**：两亲性螺旋做什么？→ 结合PA（特异性验证）
3. **机制问题**：CIDEA如何让两个脂滴靠近？→ 形成trans-complexes
4. **物理问题**：脂质如何跨膜转移？→ PA降低屏障通透性

**关键转折**：
从"CIDEA促进融合"的现象 → 到"两亲性螺旋-PA"的分子机制 → 再到"物理膜应力"的原理层面解释

## 字数要求

| 模块 | 字数 |
|------|------|
| 研究背景 | 150-200字 |
| 每个论点 | 200-300字（论点+思路+关键数据+逻辑）|
| 临床意义 | 150-200字 |
| 全文总计 | 1000-1500字 |

## 质量检查清单

- [ ] 标题体现核心发现而非泛泛主题
- [ ] 背景说明"为什么这个研究重要"
- [ ] 每个论点先说明"作者想证明什么"
- [ ] 论证步骤清晰（第一步→第二步→第三步）
- [ ] 关键数据明确支撑哪个论点
- [ ] 逻辑链条完整（A发现→推理→B结论）
- [ ] 临床意义说明"改变了什么认知"
- [ ] 延伸阅读说明与本文的关系
