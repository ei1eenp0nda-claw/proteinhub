## 🎯 核心算法亮点

### 双塔推荐模型
```
用户塔: 用户ID + 关注分布 → 64维向量
物品塔: 蛋白ID + 家族 + 描述 → 64维向量
召回:   余弦相似度 Top-K
冷启动: 家族选择 → 相似度计算
```

### 高级搜索
```python
# 多字段搜索
- 蛋白名称匹配 (权重: 1.0)
- 描述匹配 (权重: 0.3)
- 家族匹配 (权重: 0.2)

# 模糊搜索 (pg_trgm)
- 拼写容错
- 相似度排序

# 自动补全
- 前缀匹配
- 实时建议
```
